# Projeto plataforma web otimização de pedidos
Uma aplicação web desenvolvida em Python (Flask) para automatizar a correção e padronização de planilhas de pedidos comerciais. A ferramenta lê arquivos enviados pelos usuários, aplica regras de múltiplos de compra e gera automaticamente planilhas ajustadas, otimizando o fluxo entre equipes e reduzindo erros manuais.
